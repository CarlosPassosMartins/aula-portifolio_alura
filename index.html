Alura Books

O Alura Books é um projeto de aprendizado em que desenvolvi um site fictício que simula uma loja online de livros da Alura, focado em práticas de desenvolvimento web. Embora o site não permita compras reais, ele foi projetado para oferecer uma experiência de navegação completa, como se o usuário estivesse em uma loja virtual. Durante o desenvolvimento, trabalhei com HTML e CSS para criar layouts responsivos e visualmente atraentes, além de aperfeiçoar o uso de componentes reutilizáveis, menus de navegação, e organização de conteúdo para uma experiência intuitiva. Esse projeto foi fundamental para consolidar meu conhecimento em frontend e aplicar conceitos de design e usabilidade, garantindo um site funcional e esteticamente agradável, ainda que voltado exclusivamente para aprendizado.

Horta Comunitária

Desenvolvido para o programa Agrinho com o tema "Do Campo à Cidade", o site da Horta Comunitária foi idealizado para destacar a horta de Guarapuava Industrial. O projeto visa aproximar a comunidade da prática do cultivo sustentável, mostrando como a agricultura pode estar presente no cotidiano urbano.

No site, os visitantes encontram informações sobre a organização da horta, os benefícios do trabalho coletivo e como a iniciativa contribui para a valorização do meio ambiente. Além disso, ele incentiva a participação ativa da comunidade, promovendo a união e a conscientização ambiental.

Flashcards

O projeto de flashcards da Alura tem como objetivo facilitar o processo de criação de conteúdo dinâmico em sites. Em vez de repetir várias estruturas de HTML para cada cartão, o aluno aprende a utilizar JavaScript para automatizar esse processo, inserindo informações como o nome, pergunta, resposta e matéria dos cartões diretamente no código.

Essa abordagem não apenas simplifica a escrita de HTML, mas também ajuda a entender como o JavaScript pode tornar páginas mais práticas e dinâmicas. É uma experiência ideal para quem deseja aprender a trabalhar de forma mais eficiente e organizar projetos com melhor produtividade.
